Здесь находится решение дз "Нейронные сети" курса "Компьютерное зрение" от Школы Анализа Данных. \
Что реализовано: \
    • реализовал прямой и обратный проходы различных нейросетевых слоев при помощи numpy без сторонних библиотек. \
    • обучил свою полносвязную нейросеть для классификации рукописных цифр MNIST, \
    • обучил свою сверточную нейросеть для классификации изображений из набора CIFAR-10.
